# Learning Log
## Day 1 – Building My Resume Website

### Date

8 July 2026

### Objective

Build a professional resume website and prepare it for deployment to AWS.

### What I Built

- Created the project folder.
- Built the website using HTML and CSS.
- Added my professional profile picture.
- Added contact information.
- Added my technical skills.
- Added my work experience.
- Created a responsive layout for desktop and mobile devices.

### What I Learned

- Basic HTML structure
- CSS styling
- Responsive web design
- Organizing project files
- Linking CSS and images correctly

### Challenges

I had difficulty:
- Displaying my profile image.
- Opening the Git terminal.
- Understanding Git commands.
- Working with the Vim editor during Git merges.

### Solution

I learned how to:
- Correct file paths.
- Install and configure Git.
- Create my first Git repository.
- Make commits.
- Connect my local repository to GitHub.
- Push my project successfully.

### Reflection

Today I learned that Git is much more than uploading files to GitHub. It allows me to track changes, collaborate with others, and maintain version history. Although using Vim was initially confusing, I now better understand how Git workflows operate and how to recover from common issues.
